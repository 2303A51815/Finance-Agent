# Finance Agent 

An AI-powered personal finance assistant that parses bank statements, 
categorizes spending, and provides insights using Python and NLP.

## Features
- Parses CSV bank statements automatically
- Categorizes transactions (food, transport, rent, entertainment, etc.)
- Shows spending summary by category
- Built with Python, Pandas, LangChain (RAG in progress)

## Tech Stack
Python · Pandas · LangChain · OpenAI API · Streamlit (coming soon)

## Status
🚧 In Progress — RAG chain and Streamlit dashboard coming soon
